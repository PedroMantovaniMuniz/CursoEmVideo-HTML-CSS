Links externos

Para fazer um link com uma página externa, usamos a tag <a>. Exemplo:

Past lives é uma música muito bonita.

Parâmetros da tag

href=""

Sendo um parâmetro obrigatório, sua função é indicar o link para onde será o redirecionameto do usuário.

target="_blank"

Indica que o link deve ser aberto em uma nova guia.

rel=""

Indica se o link é externo.
