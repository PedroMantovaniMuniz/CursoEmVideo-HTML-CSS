Semântica em HTML5

A HTML5 é marcada por ser uma linguagem totalmente semantica, deixando tags que fazem refência à forma do site obsoletas, passando a função de cada uma totalmente para as CSS.

Podemos colocar parâmetros de CSS com tag <style> após a tag title.
