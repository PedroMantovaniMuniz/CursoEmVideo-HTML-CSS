Imagens dinâmicas

Tag <picture>

Para exibirmos imagens dinâmicas usamos a tag picture. Dentro da tag picture, marcamos a tag img com a imagem de maior tamanho.

Tag <source>

Usamos a tag source dentro da tag picture para definir as imagens alternativas.

Parâmetros da tag

media=""

Usado para definir a largura máxima em que a tela tem que estar para a imagem aparecer. Recebe o valor: (max-width: px). Deve ser colocado em ordem, sempre do menor para o maior.

srcset=""

Encontra o arquivo da imagem a ser exibida.

type=""

Indica qual o formato do arquivo da imagem.
