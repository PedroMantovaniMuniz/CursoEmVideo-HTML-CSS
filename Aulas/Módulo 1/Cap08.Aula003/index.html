Formatações adicionais

Marca texto

Nessa frase utilizamos a forma semântica de formatação textual em HTML5, com a tag <mark>

Nota: Quase toda tag em HTML5 tem o parâmetro style="", onde nós saímos da HTML e vamos para as CSS

Texto grande e pequeno

Nessa frase utilizamos a forma não semântica para formatar o texto com letras maiores em HTML5, com a tag <big>, e também utilizamos a forma semântica para formatar o texto com letras menores, com a tag <small> (Ainda não há uma explicação, porém a tag small pode ser semântica pois letras pequenas geralmente são informações importantes citadas no final de um texto)

Texto deletado

Nessa frase utilizamos a forma semântica para formatar um texto que não deve ser considerado, com a tag <del>

Texto inserido/sublinhado

Forma semântica

Nessa frase utilizamos a forma semântica para formatar um texto que deve ser enfatizado, e indicar que foi inserido depois, com a tag <ins>

Forma não semântica

Nessa frase utilizamos a forma não semântica para formatar um texto que está sublinhado, com a tag <u>

Texto sobrescrito

Pdemos utilizar a forma semântica para formatar um texto onde há um termo sobrescrito, com a tag <sup>. Exemplo: x20+3

Texto subescrito

Podemos utilizar a forma semântica para formatar um texto onde há um termo subescrito, com a tag <sub>. Exemplo: H2O
