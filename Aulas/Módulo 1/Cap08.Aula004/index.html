Mais formatações

Código-Fonte/Pré-Formatação

Podemos utilizar a forma semântica para formatar um texto onde há um código que precisa de monoespaçamento, com a tag <code>. Exemplo: documento.getElementById('teste')

Também podemos combinar a tag code à tag <pre>, fazendo com que todas as quebras de linha sejam consideradas. Exemplo:

        
num = int(input('Digite um número: '))
if num % 2 == 0:
    print(f'O número {num} é PAR.')
else:
    print(f'O número {num} é ÍMPAR.')

Citações Simples

Podemos utilizar a forma semântica para formatar um texto onde há uma citação simples, com a tag <q>. Exemplo: Uma vez um velho sábio me disse: Prefiro morrer, do que perder a vida!

Citações Completas

Podemos utilizar a forma semântica para formatar um texto onde há uma citação completa, com a tag <blockquote>. Exemplo:

Beba água, pois especialistas dizem:

É melhor uma pedra no seu caminho, do que duas no rim

Abreviações

Podemos utilizar a forma semântica para formatar um texto indicando o significado de uma abreviação, com a tag <abbr>. Exemplo: A partir do HTML5, a linguagem se tornou semântica, deixando formas para as CSS.
